# Business Document — Custom T-Shirt Platform

## 1. Business Overview

### Mô hình kinh doanh
- **Loại hình**: B2C E-commerce — Print-on-Demand
- **Sản phẩm**: Áo thun (T-shirt, Polo, Hoodie) in theo thiết kế của khách hàng
- **Quy trình**: Khách tự thiết kế online → Đặt hàng → Xưởng in sản xuất → Giao hàng tận nơi
- **Thị trường**: Việt Nam, tập trung khu vực thành thị

### Giá trị cốt lõi
1. **Cá nhân hóa**: Khách hàng tự thiết kế 100% — upload ảnh, thêm text, chọn template
2. **Đơn giản**: Editor trực quan kéo thả, không cần kỹ năng thiết kế
3. **Nhanh chóng**: Từ thiết kế đến nhận hàng trong 3-5 ngày làm việc
4. **Chất lượng**: In DTG (Direct-to-Garment) chất lượng cao, bền màu

### Đối tượng khách hàng
| Segment | Mô tả | Nhu cầu chính |
|---|---|---|
| Cá nhân | Gen Z, Millennials muốn áo unique | Tự thiết kế, đơn lẻ (1-5 cái) |
| Nhóm/CLB | Nhóm bạn, team building, class | Đồng phục nhóm, đặt số lượng 10-50 |
| Doanh nghiệp nhỏ | Shop online, thương hiệu cá nhân | Merch brand, đặt 50-200 cái |
| Sự kiện | Tổ chức event, hội thảo | Áo sự kiện, đặt nhanh, cần logo |

## 3. Vận hành

### Quy trình đơn hàng

```
Khách đặt hàng → Hệ thống gửi file thiết kế đến xưởng in
    → Xưởng in xác nhận → In sản xuất (1-2 ngày)
    → Đóng gói → Giao vận (1-3 ngày)
    → Khách nhận hàng
```

### Timeline xử lý

| Giai đoạn | Thời gian | Trách nhiệm |
|---|---|---|
| Xác nhận đơn | Tự động, real-time | Hệ thống |
| Kiểm tra file thiết kế | 1-2 giờ (giờ hành chính) | Xưởng in |
| Sản xuất (in) | 1-2 ngày làm việc | Xưởng in |
| Đóng gói + bàn giao vận chuyển | 0.5 ngày | Xưởng in |
| Giao hàng nội thành | 1-2 ngày | Đối tác vận chuyển |
| Giao hàng liên tỉnh | 2-3 ngày | Đối tác vận chuyển |
| **Tổng (nội thành)** | **3-4 ngày** | |
| **Tổng (liên tỉnh)** | **4-6 ngày** | |

## 4. Chính sách

### Đổi trả
- **Lỗi sản xuất** (in sai, in lệch, áo rách): Đổi mới miễn phí trong 7 ngày
- **Sai size**: Đổi size khác trong 3 ngày (chưa giặt, còn tag), khách chịu phí ship
- **Không hỗ trợ đổi trả**: Thay đổi ý thiết kế, không thích kết quả in

### Bảo hành
- Bảo hành chất lượng in: 30 ngày (phai màu bất thường khi giặt đúng cách)
- Giặt đúng cách: Lộn trái, giặt máy ≤30°C, không dùng chất tẩy

### Hủy đơn
- Trước khi xưởng xác nhận (trong ~2 giờ): Hủy miễn phí
- Sau khi xưởng bắt đầu in: Không hỗ trợ hủy

## 5. Yêu cầu thiết kế

### Thông số kỹ thuật in

| Thông số | Giá trị |
|---|---|
| Vùng in tối đa (T-shirt) | 30cm × 40cm |
| Vùng in tối đa (Polo) | 25cm × 30cm (nhỏ hơn do cổ) |
| Vùng in tối đa (Hoodie) | 28cm × 35cm |
| Độ phân giải file | 300 DPI |
| Kích thước export | 3000 × 3000 px |
| Format file in | PNG (nền trong suốt) |
| Màu sắc | CMYK (hệ thống tự convert từ RGB) |

### Giới hạn upload
- File formats: JPG, PNG, SVG
- Kích thước tối đa: 10MB / file
- Khuyến nghị: Ảnh ≥1000px, vector (SVG) cho logo

### Template có sẵn
- 30-50 template ban đầu
- Phân loại: Sport, Minimal, Vintage, Funny, Text-only, Seasonal
- Miễn phí sử dụng
- Khách có thể customize template (đổi text, màu, thêm ảnh)

## 6. KPI & Metrics

### Metrics chính cần tracking

| Metric | Mục tiêu (tháng đầu) | Mục tiêu (6 tháng) |
|---|---|---|
| Unique visitors | 5.000/tháng | 30.000/tháng |
| Conversion rate (visit → order) | 2% | 4% |
| Đơn hàng / tháng | 100 | 1.200 |
| AOV (Average Order Value) | 300.000₫ | 400.000₫ |
| Revenue / tháng | 30M₫ | 480M₫ |
| Return rate | < 5% | < 3% |
| Design completion rate | 30% | 50% |
| Cart abandonment rate | < 70% | < 60% |

### Funnel tracking

```
Visit Landing Page
    ↓ (60% click CTA)
View Products
    ↓ (40% select product)
Open Design Editor
    ↓ (50% complete design)  ← KEY DROP-OFF
Add to Cart
    ↓ (60% proceed to checkout)
Complete Checkout
    ↓ (85% complete payment)
Order Placed
```

### Design Editor metrics (critical)
- **Editor load time**: < 3 giây
- **Design completion rate**: % users finish design vs abandon
- **Average design time**: Thời gian trung bình trong editor
- **Template usage rate**: % đơn dùng template vs thiết kế từ đầu
- **Export success rate**: % export không lỗi

## 7. Roadmap

### Phase 1 — MVP (tuần 1-4)
- Landing page
- Product listing + detail
- Design editor (core: text, image, template)
- Cart + Checkout (COD only)
- Order processing (auto-send to print shop)
- Basic order tracking

### Phase 2 — Payment & Polish (tuần 5-6)
- VNPay integration
- Mobile responsive
- Loading states + error handling
- Email notifications (order confirmation, shipped)

### Phase 3 — Growth (tuần 7-10)
- User accounts (lưu thiết kế, lịch sử đơn)
- Template marketplace (user-contributed)
- Social sharing (share design lên Facebook/Instagram)
- Discount codes / voucher system
- Analytics dashboard (admin)

### Phase 4 — Scale (tháng 3-6)
- Thêm sản phẩm: túi tote, mũ, cốc
- Multi-vendor (nhiều xưởng in)
- API cho B2B clients
- App mobile (React Native)

## 8. Rủi ro & Giải pháp

| Rủi ro | Ảnh hưởng | Giải pháp |
|---|---|---|
| User bỏ editor vì khó dùng | Mất conversion | Template gallery, tooltips, guided tour |
| File thiết kế lỗi khi in | Đổi trả, mất chi phí | Validate trước export, preview chính xác |
| Xưởng in quá tải | Giao hàng trễ | Multi-vendor, buffer time trong SLA |
| S3 downtime | Không upload được | Retry logic, fallback thông báo |
| VNPay gateway lỗi | Mất đơn | COD fallback, retry payment |
| Canvas crash trên mobile | UX tệ | ErrorBoundary, progressive enhancement |
